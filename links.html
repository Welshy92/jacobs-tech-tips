Useful Sites & Software

Below is a list of websites and software that we would recommend any person to use on their systems. We believe most of these allow the user to have a better experience when using their systems.

  • Opera GX - A web browser that was designed for gamers first. However it contains some powerful features that all users can benefit greatly from. It is a chromium based browser. You can switch easily from Google Chrome to Opera GX as it can import all of your browsing data and saved passwords. You can also use all of the chrome extensions, on top of some extensions made specifically for itself. It also has a sidebar with integrations for things such as whatsapp, twitter, instagram and many more. It also has a feature that allows you to limit the amount the CPU, memory and bandwidth the browser uses so it doesn't grind you system to a halt like chrome often can.
  • WinRAR - A piece of software that can be used to compress files and folders. This is very useful if you have large files that you wish to transfer somewhere else or make available online. You can also password protect the archives it creates within the software to keep it secure.
  • Team Viewer - A piece of software that allows users to connect to another device that also have the software running. This can be useful if you want to access a computer remotely to perform tasks, or allowing someone to help troubleshoot a problem you are having. (DO NOT GIVE SOMEONE REMOTE ACCESS TO YOUR YOUR COMPUTER UNLESS YOU TRUST THEM.)
  • Advanced SystemCare - A piece of software that you can use to speed up or clean your computer, and protect your private data. Very simple to use and leave on in the background. IOBit, the developer, also has other software that can be installed within it to increase functionality. It contains both free and paid versions.
  • OneDrive - A website and piece of software developed by Microsoft that is used for backing up you files to the cloud. The software is usually installed onto your Windows OS by default to allow automatic backups of things such as your documents folder. The website is useful to access those files from elsewhere but you can also manually upload files to it.
  • Dropbox - An alternative cloud storage/backup to OneDrive. It also can be used through both the website or software. This is usually a better option for businesses as it offers better admin functionality.
  • Gmail - A website that is an online email application. This is one of the most common apps, alongside Microsoft's Outlook. Gmail is probably the most user friendly application and can easily be used across all of your devices. It is even more handy to have if you own an Android phone.
